# 🍽️ RecipeKeeper

**RecipeKeeper** — это десктопное приложение для хранения и управления кулинарными рецептами. Позволяет создавать, редактировать, просматривать и фильтровать рецепты с поддержкой пошаговых инструкций и фотографий.

## ✨ Возможности

- 📝 **Управление рецептами** — создание, редактирование, просмотр и удаление рецептов
- 🏷️ **Категории** — группировка рецептов по категориям (Завтраки, Обеды, Ужины, Десерты и т.д.)
- 🥕 **Ингредиенты** — список продуктов с количеством и единицами измерения
- 📸 **Фотографии** — прикрепление изображений к каждому шагу приготовления
- 🔍 **Поиск и фильтрация**:
  - Поиск по названию рецепта
  - Фильтрация по обязательным ингредиентам
  - Исключение рецептов с нежелательными ингредиентами
- 📖 **Пошаговый просмотр** — удобная навигация по шагам приготовления при просмотре рецепта
- 💾 **Локальное хранение** — все данные сохраняются в локальной SQLite базе данных

## 🛠️ Технологии

- **UI Framework**: Avalonia UI 12.0.3 (кросс-платформенный)
- **Язык**: C# / .NET 10.0
- **База данных**: SQLite (Microsoft.Data.Sqlite 10.0.9)
- **MVVM**: CommunityToolkit.Mvvm 8.4.1

## 📖 Использование

### Главное окно

- **Список рецептов** — отображает все рецепты с возможностью поиска и фильтрации
- **Поиск** — введите текст для поиска по названию рецепта
- **Фильтр** — откройте панель фильтрации для настройки ингредиентов
- **Кнопка "+"** — создать новый рецепт

### Работа с рецептом

#### Просмотр
1. Выберите рецепт из списка
2. Нажмите "Просмотр" (👁️)
3. Просматривайте ингредиенты и шаги приготовления
4. Используйте кнопки "◀" и "▶" для навигации по шагам

#### Редактирование
1. Выберите рецепт из списка
2. Нажмите "Редактировать" (✏️)
3. Измените название, описание, время приготовления, порции
4. Отредактируйте список ингредиентов (один продукт на строку)
5. Работайте со списком шагов:
   - Добавляйте новые шаги
   - Удаляйте ненужные шаги
   - Добавляйте фотографии к шагам
6. Нажмите "Сохранить"

#### Добавление фотографий к шагу
- Нажмите "📷" рядом с шагом при редактировании
- Выберите файл изображения
- Фотография привяжется к шагу

#### Удаление рецепта
1. Выберите рецепт из списка
2. Нажмите "Удалить" (🗑️)
3. Подтвердите удаление в диалоговом окне

### Фильтрация рецептов

1. Нажмите кнопку "Фильтр" (🔍) на главном экране
2. В панели фильтрации отметьте:
   - **Include** — рецепт **должен** содержать этот ингредиент
   - **Exclude** — рецепт **не должен** содержать этот ингредиент
3. Список рецептов обновится автоматически
